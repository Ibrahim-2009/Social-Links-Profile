# Social-Links-Profile

Social links profile Frontend Mentor Challenge

## 🚀 Overview

This project focuses on creating a clean profile card with social media links, emphasizing layout, styling, and interactive hover states.

## 🛠️ Built With

* HTML5: semantic structure
* CSS3: Flexbox for layout
* CSS: variables for consistent theming
* Google Fonts: Inter Font

## 🎯 Features

* Centered card layout using Flexbox
* Clean dark-themed UI design
* Interactive hover effects on buttons
* Reusable CSS variables for colors and typography
* Responsive design for smaller screens

## 📚 What I Learned

* Building structured profile card layouts
* Creating consistent button styles and hover states
* Working with dark themes and color contrast
* Improving UI with spacing and alignment
